# java-lotto-precourse

## 기능 구현 목록

### 입력 (Input)
- 구매 금액 입력
- 당첨 번호 6개 입력
- 보너스 번호 1개 입력
- 잘못된 입력(숫자가 아니거나 음수거나)이면 예외 발생 후 재입력

### 출력 (Output)
- 발행 장수 및 각 로또 번호(오름차순) 출력
- 등수별 당첨 개수 표 출력
- 수익률 출력
- 에러 메시지는 형식(예: [ERROR] ...)을 요구사항에 맞춤

### 금액 및 복권 갯수 계산 (Calculator)
- 1000원 단위 검증(0, 음수, 단위 불일치 시 예외)
- count()로 구매 장수 계산

### 로또 발행 (LottoGenerator)
- 제공된 랜덤 라이브러리를 활용하여 로또 번호 생성
- 발행된 각 로또는 오름차순 정렬
- N장 발행 시 전체 목록 반환

### 로또 (Lotto)
- 숫자 6개 검증, 오름차순/범위/중복 보장

### 당첨번호 (WinningNumbers)
- 당첨 6개 + 보너스 1개 검증(범위/중복)

### 결과 판정 (Rank)
- 일치 개수와 보너스 여부로 등수 판정
- 등수별 상금 정의

### 결과 수익률 집계 (Result)
- 등수별 개수 집계
- 총 상금 합계 계산
- 수익률 계산

### Application
- 입력 -> 로또 구매 -> 당첨 금액 계산 -> 출력 순서로 동작
